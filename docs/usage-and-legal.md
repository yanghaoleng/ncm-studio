# 使用与合规说明

NCM Studio 是一个个人用途的音乐文件整理工具，适合把自己已经拥有或有权使用的歌曲原文件整理后，拷贝到自己的离线设备里，例如游泳运动耳机、车载播放器或随身播放器。

## 推荐用途

- 整理自己的歌单文本。
- 索引网易云歌曲详情页和官方媒体外链入口。
- 处理本地已有的 `.ncm` 文件。
- 将浏览器本地转换完成的音频文件下载或打包。

## 不适合的用途

- 未经授权下载或传播音乐。
- 绕过会员、付费、DRM 或版权限制。
- 批量抓取、搬运、分享或商业化分发音乐资源。
- 违反网易云音乐或其他平台服务条款的行为。

## 关于网易云外链

复制清单中的外链格式类似：

```text
https://music.163.com/song/media/outer/url?id=00000000.mp3
```

这是网易云自己的媒体外链入口，并不是本项目破解出的最终 CDN 文件地址。打开后是否能播放或下载，取决于网易云平台、版权状态、账号状态、地区和实际网络环境。

## 隐私说明

本地 `.ncm` 文件只在浏览器中处理，不会上传到本项目服务器。AI 增强解析仅在用户配置 DeepSeek API Key 后启用，会把歌单文本发送到配置的 DeepSeek 接口用于解析。
